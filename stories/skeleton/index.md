# Usage

## Skeleton Component

```jsx
  import Skeleton from 'ui/dist/admin/skeleton'
  
  <Skeleton
    active={...props | optional}
    loading={...props | optional}
    avatar={...props | optional}
    title={...props | optional}
    paragraph={...props | optional}
    rows={...props | optional}
    size={...props | optional}
    shape={...props | optional}
    width={...props | optional}
  />
```
