# Usage

## Pagination Component

```jsx
  import Pagination from 'ui/dist/admin/pagination'

  <Pagination 
    total={...props } 
    defaultCurrent={...props | optional} 
    current={...props | optional} 
    simple={...props | optional} 
    showTotal={...props | optional} 
    showTitle={...props | optional} 
    pageSize={...props | optional} 
    disabled={...props | optional} 
    itemRender={...props | optional} 
    onChange={this.onChange}
    onShowSizeChange={this.onShowSizeChange}
  />
```
